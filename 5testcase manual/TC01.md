# TC001 - Verify My Account Dashboard Is Displayed After Successful Login

| Field | Details |
|---|---|
| **ID** | TC001 |
| **SCREEN** | My Account Page — `https://ecommerce-playground.lambdatest.io/index.php?route=account/account` |
| **TEST CASE** | Verify My Account dashboard is displayed correctly after successful login |
| **DESCRIPTION** | Verify that after a registered user logs in successfully, the system redirects the user to the **My Account** page and displays the main account sections, shortcuts, and side navigation correctly. |

---

## PRE-CONDITION

- The user has a valid registered account.
- The browser is open.
- The user is on the login page.
- The site is accessible.

---

## TEST DATA

| Field | Value |
|---|---|
| Email | `test_user@lambdatest.com` |
| Password | `Test@12345` |

---

## TEST STEPS

| Step | Action |
|---|---|
| 1 | Open the browser and navigate to `https://ecommerce-playground.lambdatest.io/index.php?route=account/login`. |
| 2 | Enter the registered email: `test_user@lambdatest.com`. |
| 3 | Enter the correct password: `Test@12345`. |
| 4 | Click the **Login** button. |
| 5 | Observe the landing page after login. |
| 6 | Verify the main sections and account shortcut items displayed on the page. |

---

## EXPECTED RESULTS

- The user is logged in successfully.
- The user is redirected to the **My Account** page.
- The page heading displays **"My Account"**.
- Main account sections such as account-related options and order-related options are displayed.
- The right-side navigation menu is visible.
- No error message is shown.
