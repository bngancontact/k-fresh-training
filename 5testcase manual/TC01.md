# TC001 - Verify My Account Dashboard Is Displayed After Successful Login

| Field | Details |
|---|---|
| **ID** | TC001 |
| **SCREEN** | My Account Page — `https://ecommerce-playground.lambdatest.io/index.php?route=account/account` |
| **TEST CASE** | Verify My Account dashboard is displayed correctly after successful login |
| **DESCRIPTION** | Verify that after a registered user logs in successfully, the system redirects the user to the **My Account** page and displays the main account sections, shortcuts, and side navigation correctly. |

---

## PRE-CONDITION

- The user has a valid registered account.
- The browser is open.
- The user is on the login page.
- The site is accessible.

---

## TEST DATA

| Field | Value |
|---|---|
| Email | `test_user@lambdatest.com` |
| Password | `Test@12345` |

---

## TEST STEPS

| Step | Action |
|---|---|
| 1 | Open the browser and navigate to `https://ecommerce-playground.lambdatest.io/index.php?route=account/login`. |
| 2 | Verify that the login page is displayed successfully. |
| 3 | Locate the **E-Mail Address** input field and enter the registered email: `test_user@lambdatest.com`. |
| 4 | Locate the **Password** input field and enter the correct password: `Test@12345`. |
| 5 | Verify that the **Login** button is visible and clickable. |
| 6 | Click the **Login** button. |
| 7 | Wait for the page to load completely after login. |
| 8 | Observe the landing page displayed after successful login. |
| 9 | Verify that the system redirects the user to the **My Account** page. |
| 10 | Verify that the page heading displays **"My Account"**. |
| 11 | Verify that the main account sections and account shortcut items are displayed correctly on the page. |
| 12 | Verify that the right-side navigation menu is visible and no error message is shown. |

---

## EXPECTED RESULTS

- The login page is displayed successfully.
- The user can enter valid registered credentials into the login form.
- The **Login** button is visible and clickable.
- The user is logged in successfully after clicking **Login**.
- The system redirects the user to the **My Account** page.
- The page heading displays **"My Account"**.
- Main account sections such as account-related options and order-related options are displayed.
- Account shortcut items are displayed correctly on the page.
- The right-side navigation menu is visible.
- No error message, warning message, or unexpected system behavior is shown.
