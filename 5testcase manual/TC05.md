# TC005 - Verify User Can Logout Successfully from My Account Page

| Field | Details |
|---|---|
| **ID** | TC005 |
| **SCREEN** | My Account Page — `https://ecommerce-playground.lambdatest.io/index.php?route=account/account` |
| **TEST CASE** | Verify user can logout successfully from My Account page |
| **DESCRIPTION** | Verify that a logged-in user can log out successfully from the **My Account** page using the **Logout** link in the side navigation, and that the system redirects the user to the **Account Logout** confirmation page with the proper logout message displayed. |

---

## PRE-CONDITION

- The user has a valid registered account.
- The user is logged in successfully.
- The user is on the **My Account** page.

---

## TEST STEPS

| Step | Action |
|---|---|
| 1 | Open the browser and navigate to the login page if the user session is not already active. |
| 2 | Log in to the site using valid registered credentials. |
| 3 | Wait until the login process is completed successfully. |
| 4 | Verify that the system redirects the user to the **My Account** page. |
| 5 | Confirm that the page heading **"My Account"** is displayed. |
| 6 | Locate the right-side navigation panel on the **My Account** page. |
| 7 | Scan the navigation options and locate the **Logout** link. |
| 8 | Verify that the **Logout** link is visible and enabled for clicking. |
| 9 | Click the **Logout** link. |
| 10 | Wait for the logout page to load completely. |
| 11 | Observe the page displayed after logout and verify the page heading and logout confirmation messages. |
| 12 | Verify that the **Continue** button is visible and clickable. |
| 13 | Click the **Continue** button. |
| 14 | Observe the page displayed after clicking **Continue**. |
| 15 | Verify that no unexpected error message is shown during the logout flow. |

---

## EXPECTED RESULTS

- The user can access the **My Account** page successfully after login.
- The page heading **"My Account"** is displayed correctly before logout.
- The right-side navigation panel is visible on the page.
- The **Logout** link is displayed in the navigation panel.
- The **Logout** link is enabled and clickable.
- After clicking **Logout**, the user is logged out successfully.
- The system redirects the user to the **Account Logout** page.
- The page heading displays **"Account Logout"**.
- The logout confirmation message is displayed: **"You have been logged off your account. It is now safe to leave the computer."**
- The additional message about the shopping cart being saved is displayed correctly.
- The **Continue** button is visible and clickable on the logout page.
- After clicking **Continue**, the system navigates to the next appropriate page successfully.
- No unexpected error message, broken layout, or system warning is shown during the process.
