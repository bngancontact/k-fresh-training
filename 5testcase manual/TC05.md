# TC005 - Verify User Can Logout Successfully from My Account Page

| Field | Details |
|---|---|
| **ID** | TC005 |
| **SCREEN** | My Account Page — `https://ecommerce-playground.lambdatest.io/index.php?route=account/account` |
| **TEST CASE** | Verify user can logout successfully from My Account page |
| **DESCRIPTION** | Verify that a logged-in user can log out successfully from the **My Account** page using the **Logout** link in the side navigation, and that the system redirects the user to the **Account Logout** confirmation page with the proper logout message displayed. |

---

## PRE-CONDITION

- The user has a valid registered account.
- The user is logged in successfully.
- The user is on the **My Account** page.

---

## TEST DATA

| Field | Value |
|---|---|
| Logout Entry | `Logout` |

---

## TEST STEPS

| Step | Action |
|---|---|
| 1 | Log in to the site using valid registered credentials. |
| 2 | On the **My Account** page, locate the **Logout** link in the right-side navigation panel. |
| 3 | Click the **Logout** link. |
| 4 | Observe the page displayed after logout. |
| 5 | Verify the logout page heading and the logout confirmation message. |
| 6 | Click the **Continue** button on the logout page. |

---

## EXPECTED RESULTS

- The **Logout** link is visible and clickable.
- The user is logged out successfully.
- The system redirects the user to the **Account Logout** page.
- The page heading displays **"Account Logout"**.
- A logout confirmation message is displayed: **"You have been logged off your account. It is now safe to leave the computer."**
- The additional message about the shopping cart being saved is displayed.
- The **Continue** button is visible and clickable.
- No unexpected error message is shown.
