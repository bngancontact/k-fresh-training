# TC002 - Verify User Can Update Account Information Successfully

| Field | Details |
|---|---|
| **ID** | TC002 |
| **SCREEN** | Edit Account Information — `https://ecommerce-playground.lambdatest.io/index.php?route=account/edit` |
| **TEST CASE** | Verify user can update account information successfully from Edit Account page |
| **DESCRIPTION** | Verify that a logged-in user can update account information such as **First Name**, **Last Name**, and **Telephone** from the **Edit Account** page by clicking the **Continue** button, is redirected back to the **My Account** page, sees a success message, and the updated data is saved successfully. |

---

## PRE-CONDITION

- The user has a valid registered account.
- The user is logged in successfully.
- The user is on the **My Account** page.
- Existing account information is available for update.

---

## TEST DATA

| Field | Value |
|---|---|
| First Name | `Ngan_Update` |
| Last Name | `Bao_Update` |
| Telephone | `0987654321` |

---

## TEST STEPS

| Step | Action |
|---|---|
| 1 | Log in to the LambdaTest E-Commerce Playground site with a valid registered account. |
| 2 | From the **My Account** page, click the **Edit Account** link or **Edit your account information** shortcut. |
| 3 | On the **My Account Information** page, update the **First Name** field with `Ngan_Update`. |
| 4 | Update the **Last Name** field with `Bao_Update`. |
| 5 | Update the **Telephone** field with `0987654321`. |
| 6 | Click the **Continue** button. |
| 7 | Observe the redirected page after submission. |
| 8 | Verify that the system redirects back to the **My Account** page. |
| 9 | Verify that a success message is displayed. |
| 10 | Open the **Edit Account** page again. |
| 11 | Verify that the updated values are saved in the corresponding fields. |

---

## EXPECTED RESULTS

- The **Edit Account** page is opened successfully.
- The user can edit the **First Name**, **Last Name**, and **Telephone** fields.
- When the user clicks the **Continue** button, the form is submitted successfully.
- The system redirects the user back to the **My Account** page.
- A success message is displayed: **"Success: Your account has been successfully updated."**
- When the user opens the **Edit Account** page again, the updated values are displayed in the corresponding fields.
- No error message is shown during the update process.
