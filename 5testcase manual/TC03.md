# TC002 - Verify User Can Change Password Successfully

| Field | Details |
|---|---|
| **ID** | TC002 |
| **SCREEN** | Change Password — `https://ecommerce-playground.lambdatest.io/index.php?route=account/password` |
| **TEST CASE** | Verify user can change password successfully from Change Password page |
| **DESCRIPTION** | Verify that a logged-in user can change the account password from the **Change Password** page by entering a new password, confirming it, clicking the **Continue** button, is redirected back to the **My Account** page, sees a success message, and the new password is saved successfully. |

---

## PRE-CONDITION

- The user has a valid registered account.
- The user is logged in successfully.
- The user is on the **My Account** page.
- The user knows the current account credentials.
- The account password can be updated.

---

## TEST DATA

| Field | Value |
|---|---|
| New Password | `Ngan@123456` |
| Confirm Password | `Ngan@123456` |

---

## TEST STEPS

| Step | Action |
|---|---|
| 1 | Log in to the LambdaTest E-Commerce Playground site with a valid registered account. |
| 2 | From the **My Account** page, click the **Change your password** link or **Password** shortcut. |
| 3 | On the **Change Password** page, enter `Ngan@123456` into the **Password** field. |
| 4 | Enter `Ngan@123456` into the **Password Confirm** field. |
| 5 | Click the **Continue** button. |
| 6 | Observe the redirected page after submission. |
| 7 | Verify that the system redirects back to the **My Account** page. |
| 8 | Verify that a success message is displayed. |
| 9 | Log out from the account. |
| 10 | Log in again using the same registered email and the new password `Ngan@123456`. |
| 11 | Verify that login is successful with the new password. |

---

## EXPECTED RESULTS

- The **Change Password** page is opened successfully.
- The user can enter a new password and confirm password.
- When the user clicks the **Continue** button, the form is submitted successfully.
- The system redirects the user back to the **My Account** page.
- A success message is displayed: **"Success: Your password has been successfully updated."**
- The user can log in successfully using the new password.
- No error message is shown during the password change process.
