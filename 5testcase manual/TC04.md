# TC004 - Verify User Can Add a New Address Successfully

| Field | Details |
|---|---|
| **ID** | TC004 |
| **SCREEN** | Add Address Page — `https://ecommerce-playground.lambdatest.io/index.php?route=account/address/add` |
| **TEST CASE** | Verify user can add a new address successfully |
| **DESCRIPTION** | Verify that a logged-in user can add a new address from the **Add Address** page by entering valid address information, clicking the **Continue** button, and being redirected to the **Address Book** page with a success message displayed. |

---

## PRE-CONDITION

- The user has a valid registered account.
- The user is logged in successfully.
- The user is on the **Address Book** section or has access to the **Add Address** page.
- No blocker prevents the user from submitting the address form.

---

## TEST DATA

| Field | Value |
|---|---|
| First Name | `Ngan` |
| Last Name | `Bao` |
| Company | `Company` |
| Address 1 | `62 Nguyen Thuong Hien` |
| Address 2 | `Address 2` |
| City | `Danang` |
| Post Code | `50000` |
| Country | `Viet Nam` |
| Region / State | `Da Nang` |
| Default Address | `No` |

---

## TEST STEPS

| Step | Action |
|---|---|
| 1 | Log in to the LambdaTest E-Commerce Playground site with a valid registered account. |
| 2 | Navigate to the **Add Address** page from the **Address Book** section. |
| 3 | Enter `Ngan` into the **First Name** field. |
| 4 | Enter `Bao` into the **Last Name** field. |
| 5 | Enter `Company` into the **Company** field. |
| 6 | Enter `62 Nguyen Thuong Hien` into the **Address 1** field. |
| 7 | Enter `Address 2` into the **Address 2** field. |
| 8 | Enter `Danang` into the **City** field. |
| 9 | Enter `50000` into the **Post Code** field. |
| 10 | Select `Viet Nam` in the **Country** dropdown. |
| 11 | Select `Da Nang` in the **Region / State** dropdown. |
| 12 | Select **No** for **Default Address**. |
| 13 | Click the **Continue** button. |
| 14 | Observe the page displayed after submission. |
| 15 | Verify the success message and the loaded address information page. |

---

## EXPECTED RESULTS

- The user can enter valid data into all required address fields.
- The **Continue** button submits the form successfully.
- The system redirects the user to the **Address Book** page after submission.
- A success message is displayed indicating that the address has been added successfully.
- The newly added address is displayed in the address information list.
- No validation error or unexpected system error is shown.
